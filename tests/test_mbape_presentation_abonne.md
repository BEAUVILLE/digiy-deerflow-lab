# Test 01 — MBAPÉ DeerFlow · Présentation abonné

## Objectif

Tester si DeerFlow peut aider un abonné DIGIYLYFE à mieux présenter son activité, sans inventer de données et sans toucher à la production.

DeerFlow ne doit pas décider.
DeerFlow ne doit pas publier.
DeerFlow ne doit pas créer de données fictives.
DeerFlow doit uniquement améliorer la présentation à partir des informations fournies.

## Mission donnée à DeerFlow

À partir des données fournies par l’abonnée, prépare uniquement :

1. une phrase courte ;
2. une description simple ;
3. un message WhatsApp client ;
4. une phrase QR ;
5. un pitch oral de 15 secondes ;
6. une liste des informations manquantes à faire valider par l’abonnée.

Contraintes :

* ne rien inventer ;
* ne pas ajouter de prix ;
* ne pas publier de numéro non validé ;
* photos en URL seulement ;
* ton humain, simple, local ;
* texte lisible sur téléphone ;
* pas de promesse excessive ;
* pas de jargon ;
* pas de blabla.
